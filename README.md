# 阅读说明
## typora说明
* 有些 markdown 语法支持需要 typora 这个软件支持。如果需要完整的阅读体验，请到 [typora官网](https://typora.io/#feature) 下载软件。
* **任何以markdown存在的文件，皆为笔记雏形**。最终形态将会以word或者pdf形式存在。因为markdown文件仅仅适合快速笔记，在排版上不如word。
* 查阅文件夹[typora配置](typora配置)，配置工具。
## obsidian说明
obsidian的好处就是动态刷新链接，我有时候要调整文件位置，为了让链接自动跟踪刷新才用的它。

## 文档内容分布和类型说明
如果在文件夹中的某个.md文件章节以第1-n章开头，有明显的引言等等内容。说明内容摘抄自GPT+豆包。如果内容上有不详细的地方，自行将它给AI，让AI完善。笔者只能说，保证自己读过，能够保证内容的稳定性，无法保证它的宽度和深度。

> 我想在这里补充一点，为什么这些文档中，有些AI的痕迹没有删除，因为它就像一个节奏点一样，会提醒读者，上一部分应该是一个比较具体的模块，要适当停下来总结，然后再进入下一部分。倒不是笔者偷懒不肯删除这部分信息。一句话形容：这些AI的额外信息就像村里跟大爷大妈讲话的嗯嗯啊啊停顿，提醒大家注意精神。

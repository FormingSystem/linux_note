# 第1章\_RCU\_专题大纲

## 1.1\_从问题到子系统再到应用

本专题按机制诞生顺序组织，而不是按术语平铺：

```mermaid
flowchart LR
    A[问题产生] --> B[抽象机制推演]
    B --> C[真实约束完善]
    C --> D[Linux RCU 子系统]
    D --> E[API 与应用]
    E --> F[验证与边界]
```

> **不可跳过的主线：** P01～P04 建立因果和子系统总图；P05～P19 完整展开 Linux 6.12.20 的主要实现路径；P20 以后才进入 API 和工程使用。
